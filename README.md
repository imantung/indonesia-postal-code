# INDONESIA POSTAL CODE

Indonesia postal code from web scraping https://kodepos.nomor.net/

## Prerequisite

- Python 3.xx
- Selenium 4.xx
- Google Chrome


## Run

```bash
python3 kelurahan.py >> csv/kelurahan_MM_YYYY.csv
python3 kota.py >> csv/kota_MM_YYYY.csv
```
